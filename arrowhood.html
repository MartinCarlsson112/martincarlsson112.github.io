The Game

Arrowhood is a 2vs2 arena type archery game, where you fight to capture the crown. The game also features parkour elements, such as jumping, lunging and climbing.

About the project

Arrowhood was developed over the course of 4 weeks as the 2nd Game Project at Futuregames. The game was developed in Unreal Engine. My focus during this project was to implement the character movement features.
